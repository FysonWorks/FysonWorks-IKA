# IKA ‚Äì FysonWorks IDE
A modern, lightweight code editor created by **Caleb / FysonWorks**.  
IKA features a clean UI, dark theme, HTML live preview, a chunk-based editor, a snippet manager, and more ‚Äî all built using Python and Tkinter.

---

##  Features

###  Modern Dark UI
IKA uses a custom dark theme with clean typography and an easy-to-read layout.

###  Multi-Language Support
- **Python**
- **HTML** (with live preview panel)

###  Live HTML Preview
As you type HTML, the preview updates automatically.

###  Chunk Editor
Break your code into pieces (chunks), edit separately, then **stitch** them together into the main editor.

###  Snippet Manager
Save commonly used code blocks and re-use them anytime.

###  Image Manager
Add images, view them in a panel, and **auto-insert paths** into your code.

###  Run Code Instantly
- Run Python directly inside the IDE  
- Open HTML in your browser  
- View all output in the built-in console
